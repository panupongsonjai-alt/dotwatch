# dotWatch Admin Firebase Service

วางไฟล์:

```txt
src/services/firebase.js
```

ไปที่:

```txt
dotwatch-admin/src/services/firebase.js
```

ถ้ายังไม่ได้ติดตั้ง Firebase:

```powershell
npm install firebase
```

จากนั้นสร้าง `.env` ใน `dotwatch-admin` โดยอ้างอิงจาก `.env.example`

ถ้าตอนนี้ยังใช้ Mock API อยู่ ให้ใช้:

```txt
VITE_USE_MOCK_ADMIN_API=true
```

# การต่อสาย Arduino UNO R3

## OLED SSD1306 0.96 นิ้ว I2C

| OLED | Arduino UNO R3 | หน้าที่ |
|---|---|---|
| GND | GND | กราวด์ |
| VCC | 5V หรือ 3.3V ตามสเปกโมดูล | ไฟเลี้ยง |
| SDA | A4 หรือขา SDA | ข้อมูล I2C |
| SCL | A5 หรือขา SCL | Clock I2C |

UNO R3 มีขา SDA/SCL ซ้ำบริเวณใกล้ AREF ซึ่งเชื่อมวงจรเดียวกับ A4/A5

## DHT11

| DHT11 | Arduino UNO R3 | หน้าที่ |
|---|---|---|
| VCC | 5V | ไฟเลี้ยง |
| DATA | D2 | ข้อมูลเซนเซอร์ |
| GND | GND | กราวด์ |

หากเป็นโมดูล DHT11 แบบ 3 ขา มักมีตัวต้านทาน pull-up มาแล้ว
หากเป็นเซนเซอร์ DHT11 แบบตัวเปล่า ให้ต่อ 4.7k-10k ระหว่าง DATA กับ VCC

## แผนผังย่อ

```text
Arduino UNO R3          OLED SSD1306
-----------------------------------
5V/3.3V  -------------> VCC
GND      -------------> GND
A4       -------------> SDA
A5       -------------> SCL

Arduino UNO R3          DHT11
-----------------------------------
5V       -------------> VCC
GND      -------------> GND
D2       -------------> DATA
```
